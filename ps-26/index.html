PICT B.E. Course Structure

First Year (FE)

  • Engineering Mathematics-I
    Foundations of calculus and linear algebra.
  • Basic Electrical Engineering
    Principles of electric circuits and machines.
  • Engineering Physics
    Fundamental physics concepts relevant to engineering.
  • Engineering Chemistry
    Chemical principles for engineering applications.
  • Programming in C
    Basics of programming using C language.

Second Year (SE)

  • Data Structures
    Organizing and managing data efficiently.
  • Digital Logic Design
    Design and analysis of digital circuits.
  • Computer Organization
    Understanding internal architecture of computers.
  • Object-Oriented Programming
    Concepts using C++ or Java.
  • Discrete Mathematics
    Mathematical structures for computer science.

Third Year (TE)

  • Operating Systems
    Concepts of process, memory, and file management.
  • Database Management Systems
    Design and use of relational databases.
  • Computer Networks
    Data communication and networking protocols.
  • Software Engineering
    Software development lifecycle and models.
  • Web Technologies
    Frontend and backend development basics.

Final Year (BE)

  • Artificial Intelligence
    Fundamentals of AI and machine learning.
  • Cloud Computing
    Virtualization, cloud models, and services.
  • Information Security
    Principles of cybersecurity and encryption.
  • Project Work
    Capstone project with real-world application.
  • Electives
    Various advanced topics as per interest.
